## 算术运算符

```js
console.log(10/4) //2.5
console.log(10%4) //2 此为取余
```

## 递增和递减

```js
let num=1
console.log(num++)//结果1
console.log(num)//结果2
```

## 比较运算符

```js
console.log(12==12)//true
console.log(12=='12')//ture
console.log(12==='12')//false
console.log(12==12)
console.log(12!='12')//false
console.log(12!=='12')//true
```
