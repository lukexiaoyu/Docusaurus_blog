## tab生成

所有的代码简写后按tab快速生成，如

p  h1  h2  div  

css 中h100  w100 bgc 等等 一般首字母

## 多个标签

html中 div*3

就会有3个div了

## class ,id

div.one   div#one

## 父子级关系

div>p

## 兄弟级关系

div+p

## 默认内容

div{one}

```html
<div>one</div>
```



## 自增

div.one$*4

```html
<div class="one1"></div>
<div class="one2"></div>
<div class="one3"></div>
<div class="one4"></div>
```
