## 单个文件监视自动编译

如有文件one.ts

输入命令

```node
tsc one.ts -w
```

就会自动生成one.js，关闭终端也就不监视了

## 全文件自动监视编译

在目录下创建tsconfig.json

然后在目录下输入命令

```
tsc -w
```

这下目录下的所有ts文件全部被监视编译了
